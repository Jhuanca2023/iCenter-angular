# Base de Datos - iCenter

## Archivos

- `schema.sql` - Esquema completo de tablas, índices, triggers y políticas RLS
- `seed.sql` - Datos iniciales (opcional)

## Instrucciones

1. Ve a Supabase Dashboard → SQL Editor
2. Copia y ejecuta `schema.sql`
3. (Opcional) Ejecuta `seed.sql` para datos de prueba
